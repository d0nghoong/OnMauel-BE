# 👵🏻👴🏻 온마을 (On-Maeul) 🧑🏻‍💻

> 어르신의 디지털 기기 사용 어려움과 청년의 유연한 수입 기회 니즈를 연결하여, 세대 간 상생 구조를 만드는 O2O 도움 매칭 플랫폼입니다.

---

## 📝 프로젝트 개요

'온마을(On-Maeul)'은 디지털 시대에 어르신들이 겪는 정보 접근성 문제와 청년들의 단기적 수입 창출 요구를 동시에 해결하기 위해 기획된 서비스입니다.

핵심 기능은 **'AI 챗봇 기반 도움 요청'**입니다. 디지털 기기 사용이 미숙한 어르신도 쉽게 접근할 수 있도록, 챗봇과의 대화를 통해 도움 요청 사항을 전달하면 GPT API를 기반으로 요청글이 등록됩니다.

청년들은 '청년 버전' 앱을 통해 실시간으로 업로드되는 도움 요청 목록을 확인하고, 스마트폰, TV, 키오스크 등 기기별 필터링 및 검색을 통해 자신에게 맞는 요청을 수락하여 매칭이 이루어집니다.

또한, 1회성 도움을 넘어 **'수업' 기능**을 제공합니다. 어르신과 청년 누구나 수업을 개설하고 수강할 수 있어 ('기초 스마트폰 수업', '열무김치 김장하기' 등), 서로의 노하우와 지식을 공유하며 장기적으로 어르신들의 디지털 접근성을 높이는 것을 목표로 합니다.

## 🗓️ 프로젝트 정보

* **개발 기간**: 2025.07.21 ~ 2025.08.21
* **팀 구성**: 백엔드 2, 프론트엔드 2, 디자이너 1
* **본인 역할 (Backend)**:
    * Spring Security 기반 회원 시스템 (로그인, 회원가입)
    * 클래스 조회 (검색, 페이지네이션) 및 신청 기능
    * 도움 요청 게시판 CRUD 및 검색 기능

## 🚀 주요 기능

* **회원 관리**: Spring Security 기반 회원가입 및 로그인, 내 정보 CRUD
* **활동 내역**: 도움 준/받은 내역, 수강/개설 클래스 내역 조회
* **클래스**: 클래스 조회 (검색/페이지네이션), 수강 신청
* **도움 요청**: 요청 게시글 CRUD, 키워드 검색
* **매칭 관리**: 청년의 요청 수락 및 매칭 상태 관리
* **AI 챗봇**: GPT API 기반 채팅 및 채팅 기반 요청글 생성
* **피드백**: 도움 완료 후 리뷰 작성
